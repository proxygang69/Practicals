PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics I
    Fundamentals of calculus, matrices, and complex numbers.
  • Engineering Physics
    Concepts of mechanics, optics, and electromagnetism.
  • Engineering Chemistry
    Study of chemical properties, bonding, and material science.
  • Basic Electrical Engineering
    Introduction to circuits, current, voltage, and basic laws.
  • Engineering Graphics
    Technical drawing and projection methods.

Second Year (SE)

  • Data Structures and Algorithms
    Introduction to stacks, queues, trees, and graphs.
  • Computer Organization
    Study of CPU, memory, and I/O organization.
  • Discrete Mathematics
    Logic, set theory, combinatorics, and graph theory.
  • Object Oriented Programming
    Concepts of classes, inheritance, and polymorphism in C++/Java.
  • Database Management Systems
    Relational databases, SQL, normalization techniques.

Third Year (TE)

  • Operating Systems
    Process management, memory management, file systems.
  • Computer Networks
    Network layers, TCP/IP, routing protocols.
  • Software Engineering
    Software development life cycle, agile methods.
  • Machine Learning
    Introduction to supervised and unsupervised learning.
  • Web Technologies
    HTML, CSS, JavaScript, backend integration basics.

Final Year (BE)

  • Distributed Systems
    Principles of distributed computing and architectures.
  • Cloud Computing
    Cloud models, virtualization, and services like AWS, Azure.
  • Artificial Intelligence
    Basics of AI, problem solving, knowledge representation.
  • Cybersecurity
    Principles of securing systems, networks, and data.
  • Project Work
    Major project development based on real-world problems.
